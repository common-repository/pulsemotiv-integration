== Description ==

#### pulsemotiv integration for WordPress

This plugin enables you to integrate audio pulses into your pages by using short codes with a valid API key generated in your Pulsemotiv account.

= Getting started =

* Go to your Pulsemotiv account and navigate to API-keys page: [pulsemotiv API keys](https://insights.pulsemotiv.com/user-keys).
* You will have a key generated for you on the API keys page. If not, click to generate a new key.
* Copy the API key and paste it into the necessary input on the plugin page.
* Click submit and your entered API key will be verified with your Pulsemotiv account.
* If you have entered a valid key, the plugin will display all of the current pulse drafts that are ready to be integrated into your page
* You can copy the short code from the list of your pulses and paste it into the page where you want to embed your pulse.
* If you are on version 4 of WordPress this plugin provides an additional dropdown for your page editor which will display your pulse drafts.
    * Place your text cursor where you want to integrate the pulse.
    * Select the pulse you want to integrate from the dropdown.
    * This will input the short code of the pulse you selected.
* Now you can view the pulse in preview mode and make changes as needed.
* Click publish or update the page for the pulse to register as live.
* Once your pulse is live, it will appear as published in the Pulsemotiv backend.
* Live pulses will no longer appear as an option in drafts.

#### WHAT IS PULSEMOTIV?

Pulsemotiv is a digital experience solution designed to keep your users more connected to your brand via conversational guidance, personalized web audio, triggered content delivery, and intelligent analytics with no coding required. To learn more go to [pulsemotiv.com](https://www.pulsemotiv.com).

== Installation ==

= Installing the plugin =
1. In your WordPress admin panel, go to *Plugins > New Plugin*, search for **Pulsemotiv Integration** and click "*Install now*"
1. Alternatively, if you have the plugin zip, upload the contents of `pulsemotiv-wp-plugin.zip` to your plugins directory, which usually is `/wp-content/plugins/`.
1. Activate the plugin
1. Set [your API key](https://insights.pulsemotiv.com/user-keys) in the plugin settings.

== Frequently Asked Questions ==

= Does changing API keys remove already embedded pulses? =

No, changing your API key won't remove your pulses from the page where they are already embedded.

= Where can I find my API key to connect to pulsemotiv? =

[You can find your API keys here](https://insights.pulsemotiv.com/user-keys)
